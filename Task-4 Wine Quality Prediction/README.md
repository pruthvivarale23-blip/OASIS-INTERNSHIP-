# Wine Quality Prediction Using Machine Learning

## Overview

This project focuses on predicting wine quality based on various chemical attributes using machine learning classification algorithms. Multiple models were trained and evaluated to determine the most effective approach for quality prediction.

## Objectives

* Analyze wine quality data and chemical characteristics.
* Perform exploratory data analysis and visualization.
* Train classification models for quality prediction.
* Compare model performance using evaluation metrics.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Models Used

* Random Forest Classifier
* Stochastic Gradient Descent (SGD) Classifier
* Support Vector Classifier (SVC)

## Key Outcomes

* Conducted data preprocessing and feature analysis.
* Built and evaluated multiple classification models.
* Compared model accuracy and performance.
* Identified important factors influencing wine quality.
