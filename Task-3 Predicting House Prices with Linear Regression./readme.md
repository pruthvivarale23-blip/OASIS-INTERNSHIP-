# House Price Prediction Using Linear Regression

## Overview

This project aims to predict house prices based on various property features using a Linear Regression model. The project demonstrates the practical application of machine learning techniques for predictive analytics.

## Objectives

* Explore and preprocess housing data.
* Identify important features affecting house prices.
* Build and train a Linear Regression model.
* Evaluate model performance using regression metrics.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Key Outcomes

* Performed data preprocessing and feature engineering.
* Trained a Linear Regression model for price prediction.
* Evaluated model performance using Mean Squared Error and R² Score.
* Visualized actual versus predicted house prices.
